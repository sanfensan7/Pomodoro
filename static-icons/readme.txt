# 备用图标目录

如果您在使用微信开发者工具时遇到图标加载问题，请将此目录下的PNG图标文件复制到项目根目录下的images文件夹中。

## 使用方法

1. 在这个目录(static-icons)中放置以下PNG图标文件：
   - clock.png
   - clock-active.png 
   - task.png
   - task-active.png
   - stats.png 
   - stats-active.png
   - settings.png
   - settings-active.png

2. 将这些文件手动复制到项目根目录的images文件夹中

3. 重启微信开发者工具

## 图标格式要求

- 尺寸：建议使用81x81像素的PNG图片
- 文件大小：每个图标不超过40KB
- 透明背景：图标应有透明背景
- 颜色：普通状态使用灰色(#999)，激活状态使用粉红色(#ff6b6b)

如果自动生成的图标问题依然存在，您也可以使用其他工具自行设计适合的图标，并按照上述文件名命名。
