# 🍅 番茄时钟 - 学习效率神器

一个功能丰富的微信小程序，集成番茄工作法、错题本管理、任务管理、学习日历、周报生成和娱乐功能，帮助用户提高学习效率和专注度。

- 📅 学习日历 - GitHub风格热力图，可视化学习历程
- 📊 每周学习报告 - 智能生成专业学习报告
- 🖼️ 海报分享功能 - 精美海报展示学习成果

## 🌟 功能特色

### 🍅 专注计时器
- **多种计时器样式**：环形、直线、极简三种风格
- **番茄工作法**：25分钟专注 + 5分钟休息的科学节奏
- **目标进度显示**：实时显示每日/每周目标完成情况
- **达成庆祝动画**：目标完成时的精美动画和震动反馈
- **智能提醒**：震动、弹窗、重复提醒多种方式
- **个性化设置**：自定义时长、主题色、提醒方式

### 🎯 目标管理系统（NEW！）

## 🧮 核心算法

### 番茄工作法算法
- **时间管理**：25分钟专注 + 5分钟短休息 + 15分钟长休息
- **目标追踪**：实时计算目标完成进度
- **自适应调节**：根据用户习惯调整时间间隔
- **智能提醒**：多种提醒方式组合

### 错题本算法
- **遗忘曲线**：基于艾宾浩斯遗忘曲线的复习提醒
- **难度评估**：根据错误次数动态调整复习频率
- **知识点关联**：智能关联相似知识点

### 统计算法
- **数据聚合**：多维度数据统计和分析
- **热力图生成**：基于专注数据生成学习热力图
- **趋势预测**：基于历史数据预测学习趋势
- **成就计算**：动态计算用户成就和等级

### 报告生成算法
- **智能分析**：分析学习模式和习惯
- **洞察生成**：基于数据生成学习洞察
- **建议推荐**：个性化学习改进建议

### 拼图游戏算法
- **A*寻路算法**：计算最优解路径
- **启发式搜索**：Manhattan距离启发函数
- **状态空间搜索**：BFS广度优先搜索

### 图片压缩算法
- **自适应压缩**：根据图片大小和场景选择压缩参数
- **质量优化**：平衡文件大小和图片质量

## 🚀 快速开始

### 📱 用户使用
1. 在微信中搜索"番茄时钟"小程序
2. 在"专注"页面开始您的第一个番茄钟
3. 在"目标"页面设定您的学习目标
4. 在"日历"页面查看学习热力图
5. 在"报告"页面生成学习周报
6. 在"错题本"页面记录和复习错题
7. 在"任务"页面管理您的学习任务
8. 在"统计"页面查看学习数据
