# 🍅 番茄时钟 - 学习效率神器

一个功能丰富的微信小程序，集成番茄工作法、错题本管理、任务管理和娱乐功能，帮助用户提高学习效率和专注度。

[![版本](https://img.shields.io/badge/版本-v1.2.2-blue.svg)](https://github.com/your-repo)
[![许可证](https://img.shields.io/badge/许可证-MIT-yellow.svg)](LICENSE)

## ✨ 核心功能

### 🎯 专注计时器
- **三种计时模式**：专注、短休息、长休息
- **多种计时器样式**：环形进度条、线性进度条、极简模式
- **自定义时长设置**：灵活配置各种时长
- **智能提醒系统**：震动、弹窗、重复提醒
- **屏幕常亮**：专注时保持屏幕亮起
- **实时统计**：显示当日专注次数和时长

### 📝 考研错题本系统
- **双模式录入**：文字模式 + 图片模式（支持拍照录入）
- **智能搜索**：支持内容、科目、标签全文搜索
- **多维筛选**：按科目、掌握程度、复习状态筛选
- **🔥 学习热力图**：GitHub风格的学习活跃度可视化
- **智能复习**：基于艾宾浩斯遗忘曲线的复习提醒
- **科目管理**：支持数学、英语、政治、专业课等科目
- **收藏系统**：重要错题标记和快速访问

### ✅ 任务管理
- **任务创建**：支持详细描述、优先级、分类
- **进度跟踪**：实时显示任务完成状态
- **多种排序**：按优先级、创建时间、完成状态排序
- **番茄钟关联**：将任务与专注时间关联
- **完成统计**：任务完成率和效率分析

### 🎮 数字拼图游戏
- **多种难度**：3×3、4×4、5×5 三种规格
- **💡 智能提示**：基于A*算法的最优路线提示
- **详细策略**：显示移动步骤和解题思路
- **成就系统**：完成统计和最佳记录
- **休息娱乐**：专注间隙的轻松游戏

### 🏆 成就系统
- **多种类型**：专注次数、时长、连击、完美专注等
- **经验系统**：完成成就获得经验值升级
- **奖励机制**：解锁新主题色和特殊功能
- **进度可视化**：清晰的成就进度展示

### � 数据统计
- **多维度统计**：日、周、月专注数据分析
- **图表可视化**：直观的数据展示
- **专注时段分析**：了解最佳专注时间
- **效率趋势**：长期学习效果跟踪

### ⚙️ 个性化设置
- **主题定制**：多种主题色彩选择
- **提醒配置**：震动、弹窗、重复次数设置
- **计时器样式**：三种不同的计时器外观
- **自动模式**：自动开始下一个时段
- **屏幕设置**：专注时保持屏幕常亮

## 🚀 快速开始

### 📱 用户使用
1. 在微信中搜索"时间领航员"小程序
2. 在"专注"页面开始您的第一个番茄钟
3. 在"错题本"页面记录和复习错题
4. 在"任务"页面管理您的学习任务
5. 在"统计"页面查看学习数据
6. 在"设置"页面个性化配置

## 💻 技术架构

### 技术栈
- **框架**：微信小程序原生框架
- **语言**：JavaScript ES6+、WXML、WXSS
- **存储**：微信小程序本地存储 API
- **开发工具**：微信开发者工具、Cursor AI

### 核心算法
- **寻路算法**：拼图游戏智能提示
- **艾宾浩斯遗忘曲线**：错题复习调度
- **热力图算法**：学习活跃度可视化
- **成就系统算法**：进度跟踪和奖励机制

### 项目结构
```
├── pages/                 # 页面文件
│   ├── focus/             # 专注计时器页面
│   ├── task/              # 任务管理页面
│   ├── mistakes/          # 错题本相关页面
│   ├── stats/             # 统计页面
│   ├── settings/          # 设置页面
│   ├── achievements/      # 成就系统页面
│   └── puzzle/            # 拼图游戏页面
├── components/            # 自定义组件
├── utils/                 # 工具函数
│   ├── mistake-manager.js # 错题本管理器
│   ├── achievement-tracker.js # 成就追踪器
│   ├── reward-system.js   # 奖励系统
│   └── vibrate.js         # 震动反馈
├── images/                # 图片资源
└── app.js                 # 小程序入口文件
```
